# Contributing

Thank you for your interest in contributing to Agent Arbiter.

## Ways to Contribute

- **Report issues** — open a GitHub issue for bugs, unclear spec language, or schema gaps
- **Propose spec changes** — open an issue first to discuss before submitting a PR
- **Improve examples** — add or refine JSON scenarios that align with the schema
- **Fix typos or clarity** — PRs for documentation improvements are welcome

## Guidelines

- Spec changes must include a rationale explaining why the change improves correctness or clarity
- Examples must validate against the schema in `schema/authority-graph.json`
- Keep pull requests focused — one change per PR

## Process

1. Open an issue to describe the change
2. Fork the repository and create a branch
3. Submit a pull request referencing the issue
