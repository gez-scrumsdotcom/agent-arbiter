# Authority Layers

> Stub — to be completed.

Authority is structured in layers. Higher layers always override lower layers.

## Layer Order

| Layer | Description |
|---|---|
| Constitutional | Regulatory and legal constraints |
| Institutional | Organizational policy |
| System | System-of-record rules |
| Agent | Agent-level preferences |

## Rules

- A higher layer can never be overridden by a lower layer
- Each layer must be explicitly scoped to a domain
- Delegation may only flow downward
